# Contributor Covenant Code of Conduct

## Our pledge

We pledge to make participation in this project a harassment-free experience for everyone, including families and contributors of all backgrounds.

## Our standards

- Be respectful and constructive.
- No harassment, trolling, or sexualized content — this project targets children's desktops.
- Accept responsibility for mistakes and focus on what is best for the community and end users' safety.

## Enforcement

Report unacceptable behavior to the maintainers via GitHub issues (private security advisories for sensitive reports when available). Maintainers may warn, edit, or ban contributors who violate this code.
